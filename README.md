## Technology
* Kotlin
* Activity host
* ViewGroup
    * RelativeLayout
    * LinearLayout
- UI Components
    - TextView
    - EditText
    - Button
- Toast to display a message
- Explicit Intent to open second Activity
- FindViewById to find UI components
- String resource for text concatenation
